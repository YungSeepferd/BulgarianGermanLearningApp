# Bulgarian-German Learning Application

A comprehensive learning platform for mastering Bulgarian and German languages, featuring vocabulary practice, grammar explanations, and cultural insights.

## Project Setup

1. Clone the repository:
   ```sh
   git clone https://github.com/yourusername/bulgarian-german-learning-app.git
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Run the development server:
   ```sh
   npm run dev
   ```

## Key Features

- **Interactive Vocabulary Practice**: Learn and test Bulgarian and German vocabulary with flashcards
- **Grammar Explanations**: Understand grammar rules with clear examples and exercises
- **Cultural Insights**: Gain understanding of cultural aspects related to each language
- **Progress Tracking**: Monitor your learning progress and improvement over time

## Development

### Running the Application

Start the development server:

Create a production build:


Preview the production build:

## Technology Stack

- Frontend: Svelte with TypeScript
- Build Tool: Vite
- Testing: Playwright and Jest
- Linting: ESLint, Stylelint, and Prettier

## Project Structure

- `/src`: Main source code
  - `/lib`: Reusable components and utilities
  - `/routes`: Application routes and pages
- `/data`: Language data files
- `/public`: Static assets
- `/tests`: Test files
