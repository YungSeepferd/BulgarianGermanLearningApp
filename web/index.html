About This Project

This application is being reimplemented from its original Rust/WASM version.

The pseudocode and data structure documentation can be found in the pseudocode/ directory.

Key Features:

  • Spaced Repetition Learning (SM-2 algorithm)
  • Bulgarian-German Vocabulary Training
  • Grammar Exercises
  • Progress Tracking

Next Steps

To implement this application in your preferred framework:

  1. Review the pseudocode in pseudocode/
  2. Set up your development environment
  3. Use the provided JSON data in web/static/data/
  4. Implement the UI components and logic
