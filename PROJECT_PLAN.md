# Project Plan

This document outlines the development milestones for the Bulgarian-German Learning App.

## Milestones

1. **Foundations**
   - [x] Set up Rust workspace with shared library and web client
   - [x] Configure continuous integration and logging

2. **Web Application**
   - [ ] Implement basic vocabulary and grammar views
   - [ ] Add spaced repetition practice modes
   - [ ] Provide audio pronunciation via Web Speech API

3. **Content Management**
   - [ ] Define data formats for vocabulary and grammar
   - [ ] Implement tools for adding and updating content

4. **Learning Features**
   - [ ] Track user progress and statistics
   - [ ] Introduce quizzes and adaptive difficulty

5. **Polish and Deployment**
   - [ ] Improve UI/UX design
   - [ ] Deploy to GitHub Pages automatically

Progress on each milestone can be tracked through issues and pull requests.
